# LinearLoader

LinearLoader is a wrapper around [LinearProgress](https://material-ui.com/api/linear-progress/), that adds extra [Typography](https://material-ui.com/components/typography/) label centered above the component to indicate the loading status.

# Structure

`LinearLoader` component exported from `Loaders` folder is represented by a single stateless react component.

```.md
├── LinearLoaderComponent.jsx # Stateless react component
└── index.js
```
