# UserProfileButton

UserProfileButton is a component used in the header of LinkedPipes Applications frontend. Contains MaterialUI Buttons with custom SVG icons, used in `UserProfile` component. Allows control over main User related settings and inbox notifications.

# Structure

`UserProfileButtonComponent` component exported from `UserProfile` folder is represented by a stateless react component and a react `PureComponent` serving as a container.

```.md
├── UserProfileButtonComponent.jsx # Stateless react component, responsible for ui
├── UserProfileButtonContainer.jsx # React PureComponent, responsible for states
└── index.js
```
