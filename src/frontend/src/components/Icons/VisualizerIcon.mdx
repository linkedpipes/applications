# VisualizerIcon

VisualizerIcon is a universal component used in multiple places in the platform. The goal is to provide an easy access to custom Visualizer icons based on `VisualizerCode` property.

# Structure

`VisualizerIcon` component exported from `Icons` folder is represented by a stateless react component.

```.md
├── VisualizerIcon.jsx # stateless react component, responsible for ui
└── index.js
```
