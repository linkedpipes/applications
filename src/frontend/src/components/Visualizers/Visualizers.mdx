# Visualizers

A folder containing supported third-party components used for data visualization.

# Structure

`Visualizers` is a folder containing various implementations of supported data visualizers. Structure is described as follows:

```.md
├── Chord/ # Folder containing Chord visualizer related Components
├── Maps/ # Folder containing Maps visualizer related Components
├── Treemap/ # Folder containing Treemap visualizer related Components
└── index.js
```

## Example

For a specific examples of individual visualizers refer to:

1. [Pigen Maps](https://github.com/mariusandra/pigeon-maps)
1. [react-chord-diagram](https://github.com/langfordG/react-chord-diagram)
1. [react-google-charts](https://github.com/RakanNimer/react-google-charts)
