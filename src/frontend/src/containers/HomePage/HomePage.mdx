# HomePage

HomePage is a container representing a `HomePage` webpage.

# Main funcionality

- Quick starting templates with sample data sources.
- Browsing current list of executed `Discovery` sessions.
- Browsing current list of executed `ETL` sessions.
- Browsing current list of executed published `SOLID` applications.

# Structure

`HomePage` component exported from `HomePage` folder is represented by a stateless react component and a react `PureComponent` serving as a container.

```.md
├── HomeController.jsx # React PureComponent, responsible for both UI and states
├── HomePage.mdx # Documentation markdown that you are currently looking at
├── children/ # Children components used within 'HomeComponent.jsx'
    ├── QuickStartComponent.jsx # Children component responsible for displaying the demo templates
    ├── PipelinesTableComponent.jsx # Children component responsible for components of Pipeline executions
    ├── DiscoveriesTableContainer.jsx # Children component responsible for states of Discoveries sessions
    ├── DiscoveriesTableComponent.jsx # Children component responsible for components of Discoveries sessions
    └── index.js
└── index.js
```
