# NotFoundPage

NotFoundPage is a container representing a `NotFoundPage` page.

# Main funcionality

- A default page being rendered on undefined routes.

# Structure

`NotFoundPage` component exported from `NotFoundPage` folder is represented by a stateless react component.

```.md
├── NotFoundPage.jsx # Stateless react component, responsible for ui
└── index.js
```
