# ApplicationPage

ApplicationPage is a container representing a `ApplicationPage` page.

# Main funcionality

- Change default folder for application configurations in SOLID.
- Toggle light or dark mode UI theme.

# Structure

`ApplicationPage` component exported from `ApplicationPage` folder is represented by a stateless react component and a react `PureComponent` serving as a container.

```.md
├── ApplicationContainer.jsx # React PureComponent, responsible for both UI and states
├── ApplicationPage.mdx # Documentation markdown that you are currently looking at
└── index.js
```
