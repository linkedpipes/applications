# SettingsPage

UserProfilePage is a container representing a `UserProfile` page.

# Main funcionality

- Change default folder for application configurations in SOLID.
- Toggle light or dark mode UI theme.

# Structure

`SettingsPage` component exported from `SettingsPage` folder is represented by a stateless react component and a react `PureComponent` serving as a container.

```.md
├── SettingsPageComponent.jsx # Stateless react component, responsible for ui
├── SettingsPageContainer.jsx # React PureComponent, responsible for both UI and states
└── index.js
```
